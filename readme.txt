An HTML compressor/minifier written for WordPress.

== Description ==

Attempts to reduce the filesize of your blog's posts and pages by removing unnecessary whitespace at the beginning and end of lines and stripping blank lines completely. `<pre>` tags are respected and their contents are left alone. However, nested `<pre>` tags may exhibit unexpected behavior. Safely minifies HTML with no change in presentation.

== Installation ==

1. Upload `wp-html-compressor.php` and `html-compressor.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Does this plugin work with WP Super Cache? =

Yes.
